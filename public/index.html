ROUTES

Principal http://localhost:9000/api

Users

GET http://localhost:9000/api/users

GET http://localhost:9000/api/users/id
Esta ruta obtiene una habitación a partir de su id

POST http://localhost:9000/api/users

PQRS

GET http://localhost:9000/pqrs/pqrs

GET http://localhost:9000/pqrs/pqrs/id
Esta ruta obtiene una PQRS a partir de su id

GET http://localhost:9000/api/users/id
Esta ruta obtiene una PQRS a partir de su fecha

GET http://localhost:9000/api/users/id
Esta ruta obtiene una PQRS a partir de su tipo

GET http://localhost:9000/api/users/id
Esta ruta obtiene una PQRS a partir de su área

POST http://localhost:9000/pqrs/pqrs

PUT http://localhost:9000/pqrs/pqrs/id
Esta ruta permite modificar cualquier dato de una PQRS a partir de su id

PUT http://localhost:9000/pqrs/edit/id
Esta ruta permite modificar el estado de una PQRS a partir de su id
